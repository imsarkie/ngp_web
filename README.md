# Reader — Website Content

This repository contains the Markdown source for the Reader app showcase website.

Reader is a premium, beautifully crafted read-later and RSS reader built with Flutter. This site is the product's public-facing showcase: brand identity, feature storytelling, and support documentation.

## Icons

App icons live in `assets/web/`. See `assets/web/README.txt` for HTML `<head>` and `manifest.json` usage instructions.
