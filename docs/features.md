# Features

Complete capability matrix for ngpocket.

## Core Product Features

- Swipe Reader with directional overlays
- Read Inbox with metadata and reading-time estimates
- Library with All, Unread, and Read filters
- RSS source management and story import
- Reader Mode with progress and source links
- Highlights capture and browsing
- Tag management with suggestions
- Typography controls and reader personalization
- Morning sync notifications with unread threshold controls
- Shared link ingestion into reading queue
- Offline-first persistence with parser fallback

## Coverage Note

This docs site covers behavior, workflows, and expected outcomes for each product area.
